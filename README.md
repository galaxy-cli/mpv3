# SETUP
* Use `chmod u+x mpv.sh` to make the script executable
* Use `./mpv.sh` to run script

# USAGE
> `./mpv3.sh -x`
> `./mpv3.sh -o FILE`
> `./mpv3.sh -e`
> `./mpv3.sh -t`

# FLAGS
-  `-x`    Use clipboard (xsel)
-  `-o`    Open and speak a file
-  `-e`    Echo user input
-  `-t`    Use tgpt output
